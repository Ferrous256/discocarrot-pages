06 / Patterns & Rules
Design Rules
Do
Use border-radius: 2px on all interactive elements
Use 1px solid var(--slate) for all borders
Use orange only for accents, never as backgrounds (except CTA)
Animate hover states: translateY(-3px) for buttons, scaleX for underlines
Keep body text at --muted, headings at --white
Use the swapped heading hierarchy (big title, small label beneath)
Don't
Use rounded corners (border-radius > 4px)
Use em dashes. Use colons, periods, or commas instead
Use emojis in professional-facing content
Use fabricated statistics or percentages
Self-attribute quotes ("Disco Carrot Philosophy")
Mix font roles (Bebas for body, Manrope for labels)
